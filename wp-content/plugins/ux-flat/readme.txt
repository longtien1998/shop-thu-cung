== Installation ==

You can install the UX Flat from your WordPress Dashboard or manually via FTP.

= From WordPress Dashboard =

1. Navigate to 'Plugins -> Add New' from your WordPress dashboard.
2. Search for `UX Flat` and install it.
3. Activate the plugin from Plugins menu.
4. Configure the plugin's settings

= Manual Installation =

1. Download the plugin file: `ux-flat.zip`
2. Unzip the file
3. Upload the`ux-flat.zip` folder to your `/wp-content/plugins` directory (do not rename the folder)
4. Activate the plugin from Plugins menu.
5. Configure the plugin's settings
